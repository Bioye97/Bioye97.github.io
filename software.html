blend

C software library for computational geometry on the regular Cartesian grid. One application is the localization of window functions in irregularly shaped domains. See Ajala & Persaud (2021) or Ajala & Persaud (2022) for the technical details regarding this tool and for a seismological application involving the estimation of surface (an)elastic deformation from earthquake aftershocks.

Modified Unified Community Velocity Model (UCVM)

Unofficially modified version of the UCVM software developed and maintained by the Southern California Earthquake Center (Small et al. 2017). UCVM embodies a non-optimization-based framework for sampling the Earth model space (e.g., Ajala et al., 2022). This modified version extends its functionality to support complex model merging using the blend library and allows additional model retrofitting. Technical details about the changes from the original version can be found in Ajala & Persaud (2021).

GQ

You wish it were your men’s health magazine. Grid Query (GQ) is a collection of programs (high-level API functions) for multiscale and multidimensional model synthesis. Stay tuned for the package release and check out the SSA meeting presentation by Ajala et al. (2025) for some details regarding development.
